
# esp iot weather

A DHT11 sensor based iot device


## Tools & Equipments


- ESP12E (ESP8266 NodeMCU)

- DHT11

- Breadboard

- Mini breadboard power supply with 5v and 3.3v output

- 9v Power adapter
